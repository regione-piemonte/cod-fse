# Componente di Prodotto

APIOPSANAURA

## Versione

1.0.0

## Descrizione

API per l'integrazione con l'anagrafica regionale degli assistiti.

## Prerequisiti di sistema

Java
Adoptium OpenJDK 11

ANT
1.8.x

Web server
Apache 2.4.x

Application server
Wildfly 23

Tipo di database
PostgreSQL 12.2

## Installazione

ant -Dtarget=tst
